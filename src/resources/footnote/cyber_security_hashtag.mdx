# #サイバーセキュリティは全員参加

👬

まさに今！サイバーセキュリティに関心を持っていただいているあなたは、サイバーセキュリティに参加しています！！

内閣サイバーセキュリティセンター(NISC)はサイバーセキュリティへの関心を高め、理解を深めていくことを狙って、
『ラブライブ！サンシャイン!!』とのタイアップ企画を実施してくれました。
その仕掛けがポスターやパンフレット...そして #サイバーセキュリティは全員参加 というハッシュタグです！さぁ呟こう！！
(多分、サイバーセキュリティ月間を過ぎても大丈夫...)
