# パスワード強度検証ツール

🔨

YouTube の動画講座内で使用されていた検証ツールは「おそらく」、security.org の "How Secure Is My Password?" だと思います。
(動画内で作成されたパスワードとその検証結果が同じでした)

なので、本アプリ上のパスワード強度検証ツールも同等の検証結果が出せるように、security.org が公開している検証モジュールを使用しています。

まるが説明している通り、パスワードの検証は端末内で行えるため、外部との通信が必要なく安心です！
